----------- Начало информации о GNOME -----------------
Полное название: GNU Network Object Model Environment (GNOME)
Описание: GNOME — это графическая оболочка для Linux, известная своим простым и интуитивно понятным интерфейсом. Поддерживает множество современных функций и расширений.
Объем дискового пространства: ~1.5 GB
----------- Конец информации о GNOME -----------------

----------- Начало информации о KDE -----------------
Полное название: K Desktop Environment Plasma (KDE Plasma)
Описание: KDE Plasma — это одна из самых настраиваемых и мощных графических оболочек для Linux. Поддерживает обширные настройки пользовательского интерфейса.
Объем дискового пространства: ~2.0 GB
----------- Конец информации о KDE -----------------

----------- Начало информации о XFCE -----------------
Полное название: XForms Common Environment (XFCE)
Описание: XFCE — это лёгкая графическая оболочка для Linux, которая потребляет мало ресурсов и подходит для старых или маломощных компьютеров.
Объем дискового пространства: ~600 MB
----------- Конец информации о XFCE -----------------

----------- Начало информации о Cinnamon -----------------
Полное название: Cinnamon Desktop Environment (Cinnamon)
Описание: Cinnamon — это графическая оболочка, разработанная для дистрибутива Linux Mint. Она предлагает традиционный рабочий стол с современными функциями.
Объем дискового пространства: ~1.2 GB
----------- Конец информации о Cinnamon -----------------

----------- Начало информации о LXDE -----------------
Полное название: Lightweight X11 Desktop Environment (LXDE)
Описание: LXDE — это ультра-лёгкая графическая оболочка, известная своей высокой производительностью и малым потреблением системных ресурсов.
Объем дискового пространства: ~300 MB
----------- Конец информации о LXDE -----------------

----------- Начало информации о MATE -----------------
Полное название: MATE Desktop Environment (MATE)
Описание: MATE — это традиционная графическая оболочка, основанная на старой версии GNOME 2. Предназначена для тех, кто предпочитает классический стиль рабочего стола.
Объем дискового пространства: ~1.1 GB
----------- Конец информации о MATE -----------------
